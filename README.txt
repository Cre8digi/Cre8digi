# CRE8DIGI - Digital Agency Website

CRE8DIGI is a modern and creative digital agency website built using HTML, CSS, JS, and Bootstrap.  
We offer services like website development, app development, video editing, SEO, UI/UX design, and branding.

## 🌐 Live Preview

[Visit Website](https:///www.cre8digi.com)

## 📁 Project Structure

- `index.html` – Home Page
- `about.html` – About Section
- `services.html` – Services Offered
- `contact.html` – Contact Form
- `css/` – Custom CSS files
- `js/` – JavaScript logic & features
- `img/` – Images used in website
- `assets/` – Fonts, icons, etc.

## 🧩 Features

- Responsive Design
- Smooth Scroll and Section Highlighting
- WhatsApp Integration in Contact Form
- Owl Carousel for Testimonials and Hero Slider
- EmailJS for form submissions (optional)

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (jQuery, Owl Carousel, WOW.js)
- Bootstrap 5
